ProjectCoursera - Static site

Hướng dẫn nhanh để chạy trang web này cục bộ:

1. Chạy bằng Node (không cần cài thêm package):

```bash
cd ProjectCoursera
node server.js
# hoặc: npm start
```

2. Mở trình duyệt vào: http://localhost:8080

3. Nếu muốn dùng Python thay thế (không cần Node):

```bash
cd ProjectCoursera
python3 -m http.server 8080
```

Ảnh và `index.html` đã có sẵn trong thư mục `images`.
